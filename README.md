# Real-Time Chat Application

A modern **Real-Time Chat Application** built with **Spring Boot** and **WebSocket**. This project demonstrates strong backend development skills, real-time communication, and secure user authentication.

---

## ✨ Features

- Real-time one-to-one and group messaging
- User registration and login with JWT Authentication
- Online/Offline user status
- Message history persistence
- Responsive and clean user interface
- Secure REST APIs

---

## 🛠️ Tech Stack

- **Backend**: Java 17, Spring Boot 3, Spring WebSocket, Spring Security, JWT
- **Frontend**: HTML5, CSS3, JavaScript, Thymeleaf
- **Database**: MySQL
- **Build Tool**: Maven
- **Others**: Spring Data JPA, STOMP Protocol

---

### Prerequisites
- JDK 17+
- MySQL Server
- Maven
